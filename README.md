# 🏋️ AI Real-Time Gym Coach

AI Real-Time Gym Coach is an intelligent fitness coaching application that uses computer vision and pose estimation to analyze workouts in real time.

The system tracks body movements through a webcam, detects exercise posture, counts repetitions, monitors workout sessions, and provides instant feedback to help users train more effectively.

This project combines AI, computer vision, and full-stack development to simulate the experience of having a virtual personal trainer.
